## PHASE 3: Alertas N8N (1.5 días)

### Workflow

```
6:00 AM México Central
    ↓
Query DB: obligaciones con deadline en 5 días
    ↓
Compose mensaje: "{obligacion} vence {deadline}. Base: ${monto}"
    ↓
POST a WhatsApp vía Evolution API
    ↓
INSERT en alerts_sent (prevent duplicates)
    ↓
Retry + Slack notify si falla
```

### Entregables

1. **alert-fiscal-deadlines.json** — Workflow N8N (300 LOC)
2. **Database migration** — CREATE TABLE alerts_sent + constraints
3. **Integration test** — Mock trigger + verify delivery

### Acceptance Criteria

- ✅ Alert delivery <5 minutos
- ✅ Zero duplicados
- ✅ Graceful fallback WhatsApp
- ✅ Timezone awareness (México Central)
